# Write-BufferToLog

## SYNOPSIS
Writes the current log buffer to log file

## SYNTAX

```
Write-BufferToLog [<CommonParameters>]
```

## DESCRIPTION
Writes the current log buffer into the log file

## EXAMPLES

### EXAMPLE 1
```
Do not use outside of Write-Log
```
