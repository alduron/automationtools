# Get-LogPath

## SYNOPSIS
Returns the current logging path

## SYNTAX

```
Get-LogPath [<CommonParameters>]
```

## DESCRIPTION
Returns the current logging path controlled by the internal config

## EXAMPLES

### EXAMPLE 1
```
Get-LogPath
```
